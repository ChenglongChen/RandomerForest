Welcome to Randomer Forests

Randomer forests are decision forests that exhibit all of the nice properties of random forests, and additionally approximate affine invariance. You can learn more about randomer forests here

You can download the repo by clicking on the tar.gz or .zip links above, or you can clone the repo by entering into the command line:

git clone https://github.com/ttomita/RandomerForest.git

Here you will find code for running randomer forest, as well as datasets for testing that it runs properly.
