Welcome to Randomer Forests

Randomer forests are decision forests that exhibit all of the nice properties of random forests, and additionally approximate affine invariance.

You can clone the repo by entering into the command line:

git clone https://github.com/ttomita/RandomerForest.git

Here you will find code for running randomer forest, as well as datasets for testing that it runs properly.
